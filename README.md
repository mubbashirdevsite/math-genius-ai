# 🌟 Overview

**Math Genius AI** is a next-generation AI-powered educational platform designed to solve mathematical problems from **basic arithmetic to advanced PhD-level mathematics**.

Students, teachers, researchers, and professionals can solve equations, generate graphs, scan handwritten problems, chat with an AI tutor, and improve learning through personalized recommendations.
